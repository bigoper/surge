Notes

Surge’s `200.html` file is a fallback, like the `404.html` file, but for single page apps and client-side routing. 

    
    my-project.surge.sh/
    ├─200.html
    ├─index.html
    ├─about.html
    └─app.js

For more details on this feature and more, please visit our docs page.
